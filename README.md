# Titanic Survival Prediction 🚢

## About this project

This is a basic machine learning project where I tried to predict whether a passenger survived the Titanic disaster or not.

I used the Titanic dataset and applied some preprocessing steps and a machine learning model to understand the data and make predictions.

---

## Dataset

The dataset contains details of passengers like:

* Age
* Gender
* Ticket class
* Fare
* Family members

Target column:

* Survived (0 = No, 1 = Yes)

---

## What I did

* Cleaned the data (handled missing values)
* Converted categorical data into numbers
* Split the data into training and testing
* Trained a model using Naive Bayes
* Checked accuracy on test data

---

## Result

The model gave around **77% accuracy**, which is decent for a beginner-level project.
